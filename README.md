# Basic Matrix Functions
<br>
ma1 and ma2 are the arrays in the form<br>
[ [ a11 , a12 , a13 ] , [ a21 , a22 , a23 ] , [ a31 , a32 , a33 ] ]<br><br>
* /add/ma1/ma2 - Addition
* /sub/ma1/ma2 - Subtraction
* /mmul/ma1/ma2 - Matrix Multiplication
* /smul/n/ma2 - Scalar Multiplication
* /tr/ma1 - Transpose
* /inv/ma1 - Inverse
* /det/ma1 - Determinant
* /co/ma1 - Cofactor
* /ad/ma1 - Adjoint
